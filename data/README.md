## Dataset Information

The dataset used for this project is not included in the repository due to size limitations.

### Data Structure

The dataset is stored as a comma-separated values (CSV) file, containing property-level housing data.

### Features Description

The dataset includes the following key variables:

* **SALE_PRC**: Sale price of the property (target variable)
* **TOT_LVG_AREA**: Total living area (square feet)
* **LND_SQFOOT**: Land area (square feet)
* **SPEC_FEAT_VAL**: Value of special features
* **structure_quality**: Structural quality rating of the property
* **age**: Age of the property

### Location-Based Features

* **LATITUDE / LONGITUDE**: Geographic coordinates
* **CNTR_DIST**: Distance to central business district
* **SUBCNTR_DI**: Distance to nearest subcentre
* **RAIL_DIST**: Distance to nearest rail line
* **HWY_DIST**: Distance to nearest highway
* **OCEAN_DIST / WATER_DIST**: Distance to water bodies

### Additional Features

* **avno60plus**: Indicator of airplane noise exposure
* **month_sold**: Month of sale

### Usage in the Project

The dataset was used to:

* perform exploratory data analysis (EDA)
* identify relationships between features and housing prices
* train and evaluate regression models
* support predictive modelling using multiple algorithms

### Note

A sample dataset can be used to reproduce this analysis, or similar housing datasets can be applied following the same workflow.
